LOGIN VIA WEB (PC/Laptop)

Langkah di Browser

  • Bersihkan cookies Netflix: buka netflix.com/clearcookies, lalu ke netflix.com.
  • Klik Sign In → masukkan email & PIN → pilih profil yang ditentukan → masukkan PIN.
  • Gagal login? Restart laptop lalu coba lagi dengan jaringan berbeda.
Login pertama disarankan pakai data/hotspot, bukan Wi-Fi rumah.
LOGIN VIA APP (HP)

Langkah di Aplikasi

  • Uninstall aplikasi Netflix → Restart HP → install ulang → login.
  • Jika Wi-Fi tidak bisa, gunakan data seluler. Jika masih gagal, coba hotspot jaringan lain.
  • Jangan spam login; ketik manual email & password (hindari copy-paste/spasi).
LOGIN VIA SMART TV

Masuk Menggunakan Web Code

  • Pada TV pilih Masuk → masukkan email → klik Masuk dari Web.
  • Scan QR dengan kamera HP atau masukkan kode yang tampil di TV.
  • Tidak perlu login Netflix di HP terlebih dahulu.
KETIKA SALAH PASSWORD / ERROR

Jangan Panik, Ikuti Ini

  • Jangan spam login; diamkan ±10 detik sebelum klik Masuk lagi.
  • Periksa & ketik ulang email/password secara manual.
  • Ganti jaringan (Wi-Fi ⇄ Data/Hotspot) atau reinstall aplikasi.
HINDARI AUTO RESET (APPS)

Prosedur Bersih Sebelum Login

  • Uninstall aplikasi Netflix.
  • Clear data & cache (sebelum uninstall jika perlu).
  • Aktifkan Mode Pesawat 2 menit → matikan.
  • Restart device, lalu ulangi lagi Mode Pesawat 2 menit → matikan.
  • Login memakai jaringan & device yang lancar.
HINDARI AUTO RESET (WEB/TV)

Langkah di Browser & Smart TV

  • Clear cache & cookies browser (Chrome/Safari/dll).
  • Restart jaringan: Mode Pesawat 2 menit lalu matikan, atau restart router/TV.
  • Login dengan jaringan stabil & device bersih; pastikan email/password benar.
  • Jangan spam login.
CATATAN PENTING

Kebijakan Penggunaan Profil

  • Satu profil hanya untuk satu pengguna, maksimal dipakai di 1 perangkat.
  • Login di dua perangkat dengan profil yang sama akan dianggap Suspicious Activity dan langganan dapat dihentikan tanpa pemberitahuan.
  • Jangan sering login-logout untuk ganti device. Pakai 1 device saja sampai masa premium berakhir; setelah itu silakan langganan akun baru di Babayshop.
  • Password bisa berubah sewaktu-waktu. Admin akan mengirim password baru jika ada perubahan (mis. masa langganan pengguna lain habis). Jika belum menerima pemberitahuan, silakan lapor ke admin.
KESALAHAN UMUM & SOLUSI

“Unable to connect to Netflix” di TV

  • Jika muncul: “Tidak Dapat Menyambung ke Netflix, Coba Lagi.” atau “Unable to connect to Netflix. Please try again.” → 100% masalah pada perangkat/TV.
  • Tunggu koneksi internet stabil atau gunakan hotspot HP.

Saat Password Dianggap Salah

  • Periksa & ketik ulang manual (hindari copy-paste & spasi tersembunyi).
  • Diamkan ±10 detik sebelum klik Masuk lagi.
  • Ganti jaringan, hapus data/cookies, atau reinstall aplikasi.
PERINGATAN

Pahami Sistem Sharing Profil

  • Akun Netflix dibeli dengan kualitas tinggi lalu dibagi per profil agar kalian bisa menikmati tayangan premium dengan harga ekonomis.
  • Karena sistemnya sharing, WAJIB memahami S&K dan mengikuti panduan agar terhindar dari error / terkunci.
